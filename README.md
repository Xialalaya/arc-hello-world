## How to deploy
1. Open [Remix IDE](https://remix.ethereum.org)
2. Connect MetaMask with Arc Testnet
3. Get test USDC from [Circle Faucet](https://faucet.circle.com)
4. Deploy the contract

Built as part of **Arc Architects Program** and preparation for **Agentic Economy on Arc Hackathon**.
